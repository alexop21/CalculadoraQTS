# CalculadoraQTS

## Linguagem
Python 3

## Frameworks
- Flask
- unittest

## Tema
Cálculos de Notas Escolares

## Operações

### Média Aritmética
- Entrada: lista de notas (float)
- Cálculo: soma / quantidade
- Saída: média

### Média Ponderada
- Entrada: notas (float), pesos (float)
- Cálculo: soma(nota * peso) / soma(pesos)
- Saída: média ponderada

### Maior Nota
- Entrada: lista de notas
- Saída: maior valor

### Menor Nota
- Entrada: lista de notas
- Saída: menor valor
